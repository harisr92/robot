# Robot
A robot which is moving on top of a table

## Place the robote on table
```shell
./bin/robot PLACE 0,0,NORTH
```

## Move robot one unit
```shell
./bin/robot MOVE
```

## Change direction to left
```shell
./bin/robot left
```

## Change direction to right
```shell
./bin/robot right
```

## Get position of robot
```shell
./bin/robot position

Output: 0,0,NORTH
```
